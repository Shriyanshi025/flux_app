# FLUX: Intelligent Crowd Orchestration Protocol

**FLUX** is a high-fidelity interactive prototype designed to orchestrate large-scale crowd movements at major venues using real-time behavioral incentives and "Frequency Lock" security protocols.

## 🚀 Vision
Built for the **Advanced Agentic Coding** challenge, FLUX replaces traditional, chaotic entry and exit flows with a precise "Green Carpet" arrival engine and a tiered "Urgency-Based" departure system.

## 🛠️ Technical Stack
- **Core Engine**: Vanilla JavaScript (ES6 Modules)
- **Visuals**: p5.js Flow-Field Kinetic Engine
- **Intelligence**: Custom 'System Brain' Proximity Proxy
- **Navigation**: Google Maps SDK (Places, Geometry, Geocoding)
- **Build System**: Vite

## 🧩 Architecture (v1.5.0 Modular)
The project follows a clean, decoupled architecture:
- `/src/pages`: Feature-specific modules (Auth, Home, Entry, Market, Exit, Profile).
- `/src/components`: Reusable UI elements (Secure QR Engine, Universal Navigation).
- `/src/services`: External integrations (Google Maps, Auth Guards, Location Logic).
- `/src/core`: Centralized State Management and Global Constants.

## 🛡️ Security Features
- **Zero-Trust Verification**: Proximity-locked QR codes that rotate every 3 seconds.
- **Biometric Identity**: Mock biometric scanning for identity pass validation.
- **HMAC-Style Rotation**: SVG-based QR payloads synchronized with server-side time-steps.
